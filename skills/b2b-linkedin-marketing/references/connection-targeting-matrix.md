# LinkedIn Connection Targeting — Keyword Matrix & Market Intelligence

## How to Build Today's 5 Keyword Combinations

### Decision Framework

1. **Start with market events** (from morning briefing): commodity prices, infrastructure investments, trade shows, policy changes
2. **Map events to buyer personas**: Who benefits? Who has budget? Who makes purchasing decisions?
3. **Check yesterday's plan**: Avoid keyword overlap — scan `~/.hermes/cron/output/*/` for the most recent plan
4. **Cover 5 distinct segments**: EPC contractors, utility companies, distributors, mini-grid developers, trade show attendees — rotate daily

### Market Intelligence → Keyword Mapping

| Market Signal | Target Persona | Keyword Direction |
|--------------|----------------|-------------------|
| Aluminum price drop (>5%) | Cost-sensitive buyers, [Competitor Name] alternatives, ABC conductor distributors | "[Competitor Name] alternative", "cost reduction power transmission", "ABC conductor fittings" |

| Policy/regulation change | Compliance officers, utility asset managers | "asset manager transmission", "grid compliance officer" |
| Competitor expansion | Buyers evaluating alternative suppliers | "line hardware alternative", "power distribution buyer" |

### Keyword Combination Patterns

Each combination follows this structure:

```
Keyword Direction: [Buyer Type] + [Product/Service] + [Intent Signal]
Industry: [1-2 LinkedIn industry categories]
Region: [Specific countries + cities, tied to active opportunities]
Titles: [3-5 decision-maker roles]
Profile Keywords: ["exact term 1" OR "exact term 2" OR "exact term 3"]
Size: [Employee range]
Exclude: [Recruiters, competitors, academia as needed]
```

### Add Note Formula

```
[Context/Observation] + [Differentiator] + [Soft CTA]

Example: "Hi [Name], with [Country]'s grid expansion accelerating, I thought you might find it useful that we supply IEC-certified preformed fittings to utility contractors across Southeast Asia and Africa. Happy to share test reports and case studies if relevant to your upcoming projects."
```

**Constraints:**
- ≤300 characters total
- Always in English for non-Chinese markets
- Never start with "We are..." or product catalog language
- Reference something specific about the prospect's role/company

## Common Mistakes

1. **Using generic terms** like "buyer" or "manager" without industry + region specificity
2. **Searching the same keywords** as yesterday — always check prior output
3. **Including recruiters/academia** — always add exclude filters
4. **Writing Add Notes that are product-first** — start with the prospect's context
5. **Ignoring market timing** — commodity price drops, trade show dates, and policy changes should drive keyword selection
6. **Targeting companies too large or too small** — EPCs need 200+ employees, mini-grid devs need 11-200, distributors need 50-500
