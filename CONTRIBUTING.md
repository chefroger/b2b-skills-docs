# Contributing

Contributions to improve the B2B Skills documentation are welcome.

## File Structure

```
b2b-skills-docs/
├── README.md          # 主文档：全景图 + 业务流程 + Skill 详解
├── QUICKREF.md        # 场景速查卡：一句话找到对应 Skill
├── REFERENCE.md       # 完整参考：每个 Skill 的技术规范
├── USAGE-ORDER.md     # 使用顺序：按阶段排列的调用指南
├── FLOWCHARTS.md      # 流程图：Mermaid 可渲染的可视化
└── CONTRIBUTING.md    # 本文件
```

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-doc`)
3. Commit your changes (`git commit -m 'Add: amazing documentation'`)
4. Push to the branch (`git push origin feature/amazing-doc`)
5. Open a Pull Request

## Documentation Standards

- All docs in Chinese (primary) with English skill names
- Use Mermaid for flowcharts
- Use tables for structured data
- Keep examples practical and copy-paste ready
- Reference actual skill names from the system
